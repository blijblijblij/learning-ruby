    class Ratios

    This class provides a few methods for calculating financial ratios. So far, three ratios are available:

    1. debt-equity ratio (der)

    2. long-term debt ratio (ltdr)

    3. total debt ratio (tdr)

def ltdr( debt=@debt, equity=@equity )
  ratio = debt/(debt+equity).to_f
  printf( "The long-term debt ratio is %.2f.\n", ratio )
  ratio
